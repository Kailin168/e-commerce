# System dependencies

- Ruby: 3.1.2
- Node: 16.17.1
- PostgreSQL: 12.12

# Configuration:

- Install packages:

  - bundle install
  - npm install --prefix client

- Database creation & initialization:

  - rails db:create db:migrate

- How to run the test suite:
  - rails s
  - npm start --prefix client
  - open [localhost:4000](http://localhost:4000/) on your browser
